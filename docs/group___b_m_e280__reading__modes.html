Definition bme280_registers.h:50

Macro for setting the mode on the BME280.

This macro generate a bitfield for use in writing to the BME280 Control Measure Setting. Use the following values for setting the mode:

  • BME280_MODE_NORMAL
  • BME280_MODE_SLEEP
  • BME280_MODE_FORCED
Parameters
modeUse the Defined BME280_MODE_XXX to assign the correct values.
Returns
Bitfield to be written into register.
