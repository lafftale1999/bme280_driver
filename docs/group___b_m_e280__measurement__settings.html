#define BME280_SET_STANDBY_MS ( ms)
Value:
((((ms) & 0x07) << 5) | BME280_CONFIG_REGISTER)

Macro for setting the standby interval. This will decide for how long the BME280 will sleep between taking measurements. (Only effective in BME280_MODE_NORMAL)

This macro generate a bitfield for use in writing to BME280 configuration register. Use the following values:

  • INTERVAL_0_5MS
  • INTERVAL_62_5MS
  • INTERVAL_125MS
  • INTEVAL_250MS
  • INTERVAL_500MS
  • INTERVAL_1000MS
  • INTERVAL_10MS
  • INTERVAL_20MS
Parameters
msUse the enum BME280_READ_INTERVALS_MS for the pre-defined values
